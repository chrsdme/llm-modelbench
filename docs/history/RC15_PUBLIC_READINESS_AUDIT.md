# LLM ModelBench 1.0.0rc15 public-readiness audit

RC15 was prepared from the pre-Git review archive before public testing.

## Corrected release blockers

- Package metadata now reads the runtime version from one source and reports `1.0.0rc15` consistently.
- Runtime OCR assets moved under `llm_modelbench/fixtures/ocr/` and are included in wheels.
- Reasoning sanity tasks use exact visible-answer scoring rather than substring presence.
- Configuration files deep-merge category weights and reject unknown fields, unknown categories, booleans, negative values, NaN, infinity, invalid URLs, and embedded URL credentials.
- Generated bytecode, personal paths, personal names, machine-specific privileged examples, and stale executable modes were removed.
- Current documentation was separated from release-history material.

## Additional public-safety work

- All-harness-error runs are marked invalid and are not added to rankings. `--strict-harness` can fail on any harness-error row.
- Executable Python, file-operation, JavaScript, and FIM tasks now require `--allow-host-code-execution`; Python and Node child processes use scrubbed environments, process-group timeout handling, and Linux resource limits. This remains a guarded host mode, not a complete sandbox.
- The read-only routing server refuses non-loopback binding unless `--allow-remote` is explicit and refuses empty/invalid summary sets unless `--allow-empty` is explicit.
- HTML reports are self-contained and no longer request Chart.js from a CDN.
- The updater rejects untracked source contamination outside known generated-evidence directories.
- CI now tests Python 3.9, 3.11, and 3.13, performs critical static checks, builds distributions, and smoke-tests the installed wheel and packaged resources.

## Remaining boundaries

- Full Git-history secret scanning must be run after this source is committed to its actual repository.
- Host-mode generated-code execution is not equivalent to a container or VM.
- Historical reasoning puzzles are sanity checks and should not be presented as contamination-resistant finals-grade reasoning measurements.
